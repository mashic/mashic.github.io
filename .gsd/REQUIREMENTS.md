# Requirements: masic.github.io Portfolio

**Defined:** 2026-03-18
**Core Value:** Demonstrate production-grade experience across 7+ business domains through clean project presentation

## v1 Requirements

### Layout & Navigation

- [ ] **NAV-01**: Fixed navbar with links to all sections
- [ ] **NAV-02**: Smooth scroll navigation to section anchors
- [ ] **NAV-03**: Active section highlighted in navbar during scroll

### Hero

- [ ] **HERO-01**: Developer name prominently displayed
- [ ] **HERO-02**: Role title (Full-Stack Developer)
- [ ] **HERO-03**: Call-to-action button scrolling to projects or contact

### About

- [ ] **ABOUT-01**: 2-3 sentence professional bio

### Skills

- [ ] **SKILL-01**: Tech stack displayed as visual badges/categories
- [ ] **SKILL-02**: Categories: Frontend, Backend, Database, DevOps, Other

### Projects

- [ ] **PROJ-01**: Project cards with name, description, tech badges
- [ ] **PROJ-02**: Links to GitHub repo and live demo per card
- [ ] **PROJ-03**: Featured projects: Kanban Board, .NET Blog, Car Rental, Hiking App
- [ ] **PROJ-04**: RSS Suite section with 7 enterprise app descriptions (no source links)

### Experience

- [ ] **EXP-01**: RSS company role and duration
- [ ] **EXP-02**: Business domains worked in (CRM, ERP, POS, Loyalty, Manufacturing, Hospitality, E-commerce, Computer Vision)
- [ ] **EXP-03**: Key patterns and technologies used

### Architecture

- [ ] **ARCH-01**: At least 2 architecture/pattern diagrams
- [ ] **ARCH-02**: Diagrams show real patterns from production work (sanitized)

### Contact

- [ ] **CONTACT-01**: LinkedIn profile link
- [ ] **CONTACT-02**: GitHub profile link
- [ ] **CONTACT-03**: Email link

### Responsiveness & Deploy

- [ ] **RESP-01**: Responsive layout at 375px, 768px, 1024px, 1440px
- [ ] **DEPLOY-01**: Builds and deploys to GitHub Pages from docs/ folder

## v2 Requirements

### Enhancements

- **ANI-01**: Section reveal animations on scroll
- **ANI-02**: Subtle hover effects on project cards
- **SEO-01**: Meta tags for social sharing (Open Graph)

## Out of Scope

| Feature | Reason |
|---------|--------|
| Blog | Maintenance burden, not core portfolio value |
| Contact form | Requires backend, use direct links |
| Theme switcher | Single polished theme sufficient |
| Multi-language | English-only audience |
